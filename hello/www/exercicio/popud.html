Atributo Objetivo
data-dismissibler True false Se definido para fechamento falsos impede pop-up clicando fora do contexto ou pressionando a tecla Esc.
Data-history true | false - Define se o pop-up cria um item de histórico do navegador quando se abre. Se definido como false, não vai criar um item de história, e, assim, não ser closeable via botão "Voltar" do navegador.
Data-overlay-theme Encarte de A /Z
Data-position-to origem | janela | seletor - "origem" indica que o pop-up deve ser centrado sobre o link que abre."Janela" indica que ele deve ser centralizado na janela do navegador. Se você der um seletor jQuery, será centrado sobre o primeiro elemento retornado pelo seletor, se esse elemento é visível. Caso contrário, ele será centralizado na janela do navegador.
Data-shadow True False
Data-url url - Valor para atualizar o URL, em vez da URL utilizada para solicitar a página
Data-theme Encarte de A Z
